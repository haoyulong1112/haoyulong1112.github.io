    # git合并到主分支上

    1. 切换到主分支master
2. git merge [name] 合并到主分支
3. git push 提交
4. php 登录服务器
5. 拉取代码
6. exit
7. 前端上线时修改fis-conf
8. qa 切换到线上地址
9. 执行fis3 release qa -w -r home
10. 编译后fis-conf改回
11. 进后台管理页面配置页面接口和权限
12. git切回到开发分支
